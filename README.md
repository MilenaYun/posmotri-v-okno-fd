# Посмотри в окно
## Описание проекта
В этой проектной работе в отличии от первой была работа с чужим кодом. Основная задача быда в стилизации CSS. В работе были спользованы следующие технологии
* Flexbox для построения сеток и выравнивания элементов
* CSS позиционирование
* Псевдоклассы
* Псевдоэдементы
* Работа с интерактивным состоянием элементов
* Pixel Perfect сверка с макетом.
